## 設計理念

### 為什麼選擇這些顏色?

1. **草地綠** - 網球最經典的場地就是草地球場(如溫布頓),綠色代表專業和傳統
2. **網球黃** - 網球的標誌性顏色,鮮明且充滿活力
3. **白色** - 網球傳統服裝顏色,代表純淨和專業

### 參考來源

主要參考了以下網球賽事的視覺設計:
- 溫布頓網球錦標賽(綠色草地)
- 美國網球公開賽(藍色硬地)
- 法國網球公開賽(紅土)

結合 [Liwei Cup](https://liwei-cup.com/) 的設計風格:
- 大膽的配色
- 球員小卡展示
- 激勵人心的標語
- 現代化的 UI

## 技術亮點

### CSS 變數系統
使用 CSS 變數(`:root`)統一管理顏色,方便未來調整:

```css
:root {
    --tennis-green: #2d5016;
    --tennis-yellow: #d4af37;
    /* ... 更多變數 */
}
```

### 響應式設計
- 桌面: 多欄佈局
- 平板: 2欄佈局
- 手機: 單欄佈局 + 漢堡選單

### 動畫效果
- 導航欄滾動陰影
- 卡片懸停 3D 效果
- 滾動淡入動畫
- 平滑滾動

## 使用方式

### 開啟網站
直接用瀏覽器開啟 `index.html` 即可預覽。

### 修改球員資料
編輯 `script.js` 的 `playersData` 陣列:

```javascript
{
    name: "你的名字",
    number: 99,
    role: "你的角色描述",
    quote: "你的霸氣宣言",
    emoji: "🎾"  // 可用任何 emoji
}
```

### 修改配色
如果你想調整顏色,編輯 `styles.css` 的 `:root` 區塊。

### 新增圖片
將圖片放入 `images/` 資料夾:
- `hero-bg.jpeg` - 主視覺背景
- `logo.jpeg` - Logo
- `photo1-6.jpeg` - 比賽照片
