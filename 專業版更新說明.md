## 💻 技術實現

### 使用的技術棧

```html
<!-- 動畫庫 -->
<script src="GSAP 3.12.2"></script>
<script src="ScrollTrigger"></script>
<script src="Vanilla Tilt.js 1.8.1"></script>
<script src="AOS 2.3.1"></script>
```

### 核心功能

1. **視差滾動 (Parallax)**
   ```javascript
   gsap.to(parallaxBg, {
       y: () => window.innerHeight * 0.5,
       scrub: true
   });
   ```

2. **3D 傾斜 (Tilt)**
   ```javascript
   VanillaTilt.init(element, {
       max: 15,
       speed: 400,
       glare: true
   });
   ```

3. **數字動畫 (Counter)**
   ```javascript
   gsap.to(number, {
       textContent: target,
       snap: { textContent: 1 }
   });
   ```

4. **滾動觸發 (ScrollTrigger)**
   ```javascript
   ScrollTrigger.create({
       trigger: element,
       start: "top 80%",
       toggleActions: "play none none reverse"
   });
   ```

## 📊 性能優化

### 已實現的優化

1. **懶加載 (Lazy Loading)**
   - 圖片進入視窗才載入
   - 減少初始載入時間

2. **請求動畫幀 (RAF)**
   - 使用 requestAnimationFrame
   - 確保 60fps 流暢度

3. **節流 (Throttle)**
   - 滾動事件節流
   - 視窗調整事件節流

4. **硬體加速**
   - 使用 transform 和 opacity
   - 觸發 GPU 加速

## 🎨 自訂設定

### 修改配色

編輯 `styles.css` 的 `:root` 區塊:

```css
:root {
    --tennis-green: #2d5016;
    --tennis-yellow: #d4af37;
    /* ... 更多顏色 */
}
```

### 修改球員資料

編輯 `script.js` 的 `playersData` 陣列:

```javascript
{
    name: "球員姓名",
    number: 1,
    role: "角色描述",
    quote: "個性宣言",
    emoji: "🎾"
}
```

### 修改動畫速度

在 `script.js` 中調整 GSAP 的 `duration` 參數:

```javascript
gsap.to(element, {
    duration: 1, // 秒數
    ease: "power2.out"
});
```

## 🐛 常見問題

### Q: 為什麼看不到 3D 效果?

**A:** 確認:
1. 瀏覽器支援 CSS 3D transforms
2. JavaScript 已啟用
3. 動畫庫已正確載入(檢查 Console)

### Q: 圖片不顯示?

**A:** 檢查:
1. 圖片檔名是否正確(區分大小寫)
2. 圖片是否在 `images/` 資料夾
3. 圖片格式是否正確(.jpg/.jpeg/.png)

### Q: 動畫卡頓?

**A:** 優化:
1. 壓縮圖片檔案大小
2. 使用現代瀏覽器(Chrome/Edge/Firefox)
3. 關閉其他佔用資源的程式

### Q: 手機版選單無法開啟?

**A:** 確認:
1. JavaScript 已載入
2. 點擊漢堡圖示(三條線)
3. 檢查 Console 是否有錯誤
