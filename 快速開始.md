  - spirit1.jpg

  - spirit3.jpg

下載 6 張 → 重新命名為:

  - photo6.jpeg

## 🎨 快速自訂

### 修改比賽日期

編輯 `index.html` 第 190-193 行:

```html
<span class="info-number animated-number" data-target="12">0</span>
<span class="info-unit">月</span>
<span class="info-number animated-number" data-target="13">0</span>
<span class="info-unit">日</span>
```

修改 `script.js` 第 230 行:

```javascript
const targetDate = new Date('2025-12-13T10:00:00').getTime();
```

---

### 修改比賽地點

編輯 `index.html` 第 209-211 行:

```html
<div class="card-main-info location-info">
    <span class="location-name">華中網球場</span>
</div>
<p class="card-sub-info">新北市三重區華中街38號</p>
```

---

### 修改球員資料

編輯 `script.js` 第 2-87 行的 `playersData` 陣列:

```javascript
{
    name: "你的名字",
    number: 99,
    role: "你的角色描述",
    quote: "你的霸氣宣言",
    emoji: "🎾"  // 可以換成任何 emoji
}
```

---

### 修改配色

編輯 `styles.css` 第 7-17 行:

```css
:root {
    --tennis-green: #2d5016;        /* 深綠色 */
    --tennis-light-green: #4a7c2c;  /* 淺綠色 */
    --tennis-grass: #7cb342;        /* 草地綠 */
    --tennis-yellow: #d4af37;       /* 金黃色 */
    --tennis-bright-yellow: #ffeb3b;/* 亮黃色 */
}
```

## 🐛 常見問題快速解決

### 問題 1: 圖片不顯示

**解決方案:**
```bash
1. 檢查圖片檔名是否完全一致(區分大小寫)
2. 確認圖片在 images/ 資料夾內
3. 重新整理瀏覽器 (Ctrl + F5)
```

---

### 問題 2: 3D 效果不動

**解決方案:**
```bash
1. 開啟瀏覽器 Console (F12)
2. 查看是否有錯誤訊息
3. 確認網路連線正常(動畫庫需要從 CDN 載入)
4. 使用 Chrome 或 Edge 瀏覽器
```

---

### 問題 3: 動畫卡頓

**解決方案:**
```bash
1. 壓縮圖片 (使用 TinyPNG)
2. 關閉其他佔用資源的程式
3. 使用較新的電腦/瀏覽器
```

---

### 問題 4: 手機版選單無法開啟

**解決方案:**
```bash
1. 確認 JavaScript 已載入
2. 檢查 Console 是否有錯誤
3. 清除瀏覽器快取
4. 重新載入頁面
```

## 📱 手機測試

### 在手機上測試

#### 方法 A: 使用 Live Server
```bash
1. 開啟 Live Server
2. 查看終端機顯示的 IP 位址 (例如: 192.168.1.100:5500)
3. 手機連接同一個 WiFi
4. 在手機瀏覽器輸入該 IP 位址
```

#### 方法 B: 上傳到伺服器
```bash
1. 使用 GitHub Pages / Netlify / Vercel
2. 上傳所有檔案
3. 獲得網址後在手機開啟
```
